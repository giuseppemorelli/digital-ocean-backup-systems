# Digital Ocean Backup System

Script to create snapshots and copy in different region of Digital Ocean

## Requirements

- nodejs >= 14
- npm >= 6

## Installation

```
npm install
cp .env.dit .env

# copy your Digital Ocean API KEY on API_KEY variable
```

## Start

```
npm start
```
