# Digital Ocean Backup System

Script to create snapshots and copy in different regions of Digital Ocean Datacenter

## Requirements

- nodejs >= 14
- npm >= 6
- Digital Ocean API Token

## Installation

```
npm install
cp .env.dit .env

# copy your Digital Ocean API KEY on API_KEY variable
```

## Start

```
npm start
```
